Linklər olan düymələrin hamısının eyni enlikdə olmasını təmin etmək üçün CSS-ə müəyyən bir enlik dəyəri təyin etmək lazımdır. Bunun üçün CSS-də `width` xüsusiyyətindən istifadə edə bilərsiniz. Məsələn, hər bir düymənin genişliyini 250px təyin edə bilərsiniz. Aşağıdakı kodu mövcud CSS hissəsinə əlavə edə bilərsiniz: ```css .button { width: 250px; margin-bottom: 15px; text-align: center; } ``` Bunu etdikdən sonra bütün düymələr eyni enlikdə olacaq və sətir dəyişiklikləri bir-birinə uyğun olacaq.
